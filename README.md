# TextAdvanture  
Zum Ausführen, bitte in VS Code öffnen und einen Live Server starten (./display/index.html).  

# Commands  
-attack(a) {who}  
-clear  
-commands(c)  
-description(d  
-drop(dr) {object}  
-inventory(i)  
-look(l) ?{object}  
-take(t) {object}  
-talk {who}  
-walk(w) {where}  

# How to play  
bsp:  
attack rainer  
talk rainer  
look  
look rainer  
t schwert  
take schwert  
c
